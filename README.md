# Illustrator Scripts Translate

Adobe Illustrator 脚本工具集，包含多个实用工具和中文化翻译版本。

## 主脚本
- `batch-translate-v2.jsx` - 批量翻译脚本（v2）
- `batch-translate.jsx` / `batch-translator.jsx` - 批量翻译
- `complete-translation.jsx` - 完整翻译脚本
- `illustrator-all-in-one.jsx` - 一体化工具
- `illustrator-navigator-final.jsx` - AI 导航工具（最终版）
- `illustrator-navigator.jsx` - AI 导航工具

## jsx/其他工具
- RandomScribble / RandomStrokeWidth / Rescale / ResizeToSize
- StrokesWeightDown / StrokesWeightUp
- TriangleMaker / TrimMasks / TrimOpenEnds

## jsx/图层管理
- ExtractFromGroup / ExtUngroup

## 使用
文件 > 脚本 > 其他脚本...
